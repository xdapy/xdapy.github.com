Operators¶

Defines some operators for filter expressions.

The expressions can be used in a query context to restrict the values of an entity’s parameter.

> mapper.find_all(Observer, filter={“age”: gt(20)})

restricts the result to all Observers with oberserver.params["age"] > 20.

xdapy.operators.ge(v)[source]¶

Greater or even than.

ge(v)(t) == t >= v

xdapy.operators.gt(v)[source]¶

Greater than.

gt(v)(t) == t > v

xdapy.operators.le(v)[source]¶

Lesser or equal than.

le(v)(t) == t <= v

xdapy.operators.lt(v)[source]¶

Lesser than.

lt(v)(t) == t < v

xdapy.operators.between(v1, v2)[source]¶

Between.

between(v1, v2)(t) == t >= v1 and t <= v2

xdapy.operators.eq(v)[source]¶

Equal.

eq(v)(t) == (v == t)

xdapy.operators.like(v)[source]¶

Like.

like(v)(t) == t.like(v)
